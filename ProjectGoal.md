The goal with this project is to build a 3D, web-based piano that allows users to visualize the keyboard and play it in real-time.

Ideally, this would be used as a portable tool to practice and improvise ideas, with the stronger visual feedback than a typical 2D piano roll.
I can also see this being used viably as an educational tool prior to committing to the purchase of a synth or piano.

Stack choice:
Tone.js: Allows real-time audio in the browser.
R3F: Allows for the construction of a 3-Dimensional keyboard.
Tailwind: Obvious. Makes styling easy.
Hosting: GitHub Pages for now.


Possible ideas stemming from this, moving forward:
- Allow for a miniture playback feature (need to study this idea more; midi, recording, etc.).
- Allow the piano role to play a selection of diverse sounds (web synthesizer with different audio effects).
- Basic user accounts that save previous sessions.
- If project reaches a level of success, perhaps porting it over to mobile/React-Native could be an option.
